# ZLAI

> 适配中文大模型的`Langchain-Agent`

## 简介

- `LLMs`: 提供远程、本地、预训练等大模型服务
- `Embedding`: 提供远程、本地、预训练等向量化模型服务
- `prompt`: 提供prompt组织流程
- `elasticsearch`: 提供elasticsearch工具
- `agent`: 提供agent工具
